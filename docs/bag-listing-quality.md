# Bag listing quality bar

- **Image:** Min 800×600, landscape preferred; use in-app upload (mobile + web).
- **Title:** Specific (e.g. “Evening pastry box”), not “Bag”.
- **Pickup window:** Must end after start; default 2h if unset.
- **Quantity:** Honest count; sold-out auto-updates after Phase 0 inventory trigger.
- **Pricing:** Rescue price below retail estimate.

Optional: first bag per merchant flagged for admin review (`merchants` notes).
