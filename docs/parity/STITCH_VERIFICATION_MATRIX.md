## Status definitions

| Status | Meaning |
|---|---|
| **Ship** | RN route + intended behaviour, **layout/typography/color/components are an effective pixel match** of the Stitch HTML for the platform-appropriate composition (mobile main canvas). |
| **Partial** | RN route exists and matches the **gross composition** (sections, ordering, primitives, copy) but **at least one of**: (a) a visible visual flourish is missing (animation, raster illustration, brand artwork), (b) a `_2` / `_refined` / `_dark` variant exists in Stitch that RN doesn't render distinctly, (c) the screen reuses a Material icon where Stitch uses a brand/Phosphor glyph, or (d) a button/action is a stub. The per-row note enumerates which. Most Partial rows correspond to **intentional native adaptations** (no desktop drawer, no HTML bottom nav, no marketing chrome) and are visually equivalent at launch. |
| **Web-only** | No RN surface — this artifact is the PWA's responsibility (marketing landing pages). |
| **Deferred** | Intentionally not implemented as a separate RN surface — usually replaced by an OS-level equivalent (e.g. OS splash instead of brand loading animation). |
